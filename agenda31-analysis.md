# Análise do Layout agenda31.com.br

## Cores Principais
- Fundo: Preto (#000000)
- Texto: Branco/Cinza claro
- Destaques: Vermelho (#FF0000) para badges e botões
- Gradiente: Preto para transparente

## Header
- Fundo preto
- Logo dourada/amarela à esquerda
- Navegação horizontal centralizada: HOME, MODELOS, CIDADES, VÍDEOS, ÁUDIOS, ATUALIZAÇÕES, SERVIÇOS, BLOG
- Botão "ANUNCIE AQUI" em vermelho à direita
- Ícones de redes sociais (Facebook, Twitter) no canto superior direito

## Banner Principal (Slider)
- Altura: ~550-600px
- Foto grande em formato paisagem (horizontal)
- Overlay com gradiente escuro
- Informações sobre a modelo no lado esquerdo inferior:
  - Nome em destaque
  - Idade, altura, peso
  - Cidade/região
  - Telefone
  - Badge de destaque (ex: "ALTO PADRÃO", "EXPERT EM ANAL")
- Estrelas de avaliação
- Ícones de ação (WhatsApp, compartilhar)

## Stories
- Seção horizontal com scroll
- Círculos com fotos das modelos
- Nome abaixo de cada círculo
- Borda colorida (gradiente ou cor sólida)

## Grid de Perfis (Cards)
- Layout em grid responsivo (3-4 colunas)
- Cada card contém:
  - Foto principal (vertical/retrato)
  - Badge de destaque no topo (ex: "FAZ VÍDEO CHAMADA", "LUXUOSA")
  - Nome
  - Idade
  - Altura e peso
  - Cidade/bairro
  - Telefone
  - Botão "Compartilhe"
  - Estrelas de avaliação

## Seção VIP
- Título "VIP'S BELO HORIZONTE"
- Grid similar aos cards normais
- Destaque visual diferenciado

## Cores para Adaptar ao farialover
- Vermelho → Roxo (#8B00FF)
- Badges vermelhas → Roxo/Rosa gradient
- Manter fundo preto
- Manter texto branco/cinza claro
